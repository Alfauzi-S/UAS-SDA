## 📝 Deskripsi Proyek

Proyek ini merupakan tugas akhir semester (UAS) mata kuliah Struktur Data dan Algoritma yang mencakup tiga komponen utama:
1. Implementasi penelusuran Binary Search Tree (BST)
2. Representasi Graph menggunakan Adjacency Matrix
3. Representasi Graph menggunakan Adjacency List

Setiap komponen telah diimplementasikan dengan bahasa pemrograman C++ sesuai dengan petunjuk yang diberikan.

---

## 🔍 Soal dan Implementasi

### 1. Penelusuran Binary Search Tree

Kami membangun BST dengan 11 simpul dan mengimplementasikan 4 jenis penelusuran:
- **Preorder**: Root → Kiri → Kanan
- **Inorder**: Kiri → Root → Kanan
- **Postorder**: Kiri → Kanan → Root
- **Level Order (BFS)**: Berdasarkan tingkat kedalaman

### 2. Representasi Graph dengan Adjacency Matrix

Kami mengimplementasikan 4 jenis graph menggunakan matriks adjacency:
- **Graph tak berarah tak berbobot**: Matriks simetris dengan nilai 0/1
- **Graph tak berarah berbobot**: Matriks simetris dengan nilai bobot
- **Graph berarah tak berbobot**: Matriks tidak simetris dengan nilai 0/1
- **Graph berarah berbobot**: Matriks tidak simetris dengan nilai bobot

### 3. Representasi Graph dengan Adjacency List

Kami mengimplementasikan 4 jenis graph menggunakan linked list:
- **Graph tak berarah tak berbobot**: Setiap simpul menyimpan daftar tetangganya
- **Graph tak berarah berbobot**: Setiap simpul menyimpan pasangan (tetangga, bobot)
- **Graph berarah tak berbobot**: Setiap simpul hanya menyimpan tujuan arahnya
- **Graph berarah berbobot**: Setiap simpul menyimpan pasangan (tujuan, bobot)
